# 個人ホームページ

Cloudflare Pagesで運用する個人ホームページのリポジトリです。

## 概要

このリポジトリは、自分用のホームページを構築・運用するためのプロジェクトです。
Cloudflare Pages上でホスティングし、高速でセキュアなサイトを目指します。

## 技術スタック

- **フロントエンド**: Vanilla HTML/CSS/JavaScript
  - フレームワークレスでシンプルな構成
  - 依存関係なしで高速な読み込み
  - 最小限の構成でメンテナンス性を確保

## セットアップ

### ローカル開発

このプロジェクトは静的HTMLなので、特別なビルドツールは不要です。

#### 方法1: ファイルを直接ブラウザで開く（最もシンプル）

1. `index.html` ファイルをブラウザで開く
   - Finderで `index.html` をダブルクリック
   - または、ブラウザにドラッグ&ドロップ
   - または、ブラウザで `ファイル → 開く` から選択

**メリット**: すぐに確認できる  
**デメリット**: 
- 一部の機能（CORSなど）が制限される場合がある
- `file://` プロトコルで動作するため、相対パスの挙動が異なる場合がある

#### 方法2: ローカル開発サーバーを使う（推奨）

**Python 3を使う場合:**

```bash
# プロジェクトディレクトリに移動
cd /Users/aj/Documents/MyHomepage

# ローカルサーバーを起動（ポート8000）
python3 -m http.server 8000

# ブラウザで http://localhost:8000 にアクセス
```

**Node.jsを使う場合:**

```bash
# プロジェクトディレクトリに移動
cd /Users/aj/Documents/MyHomepage

# serveパッケージを使用（初回のみ自動インストール）
npx serve

# または特定のポートを指定
npx serve -p 3000
```

**メリット**: 
- 本番環境に近い形で確認できる
- CORSの制限が少ない
- 相対パスが正しく動作する

#### 方法3: VS CodeのLive Server拡張機能（おすすめ）

1. VS Codeで `Live Server` 拡張機能をインストール
2. `index.html` を開く
3. 右下の「Go Live」ボタンをクリック、または右クリックメニューから「Open with Live Server」

**メリット**: 
- 自動リロード機能（ファイル保存時に自動更新）
- 非常に簡単
- VS Codeの統合

#### 方法4: その他のローカルサーバーツール

- **PHP**（インストール済みの場合）:
  ```bash
  php -S localhost:8000
  ```

- **Ruby**（インストール済みの場合）:
  ```bash
  ruby -run -e httpd . -p 8000
  ```

#### 開発の流れ

1. ローカルサーバーを起動（方法2または3を推奨）
2. ブラウザで `http://localhost:8000` を開く
3. HTML/CSS/JSファイルを編集
4. ブラウザをリロード（またはLive Serverなら自動リロード）
5. 変更を確認

#### ヒント

- **ブラウザの開発者ツール**（F12 / Cmd+Option+I）を使ってリアルタイムでCSSを編集して確認できます
- **ブラウザのキャッシュ**をクリアしたい場合は、`Cmd+Shift+R`（Mac）または `Ctrl+Shift+R`（Windows）でハードリロード

## デプロイ

Cloudflare Pagesへの自動デプロイが設定されています。
`main`ブランチへのプッシュで自動的にデプロイされます。
